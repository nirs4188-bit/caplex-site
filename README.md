# Caplex Equity Website (Netlify)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Netlify
- New site from Git OR drag-and-drop the `dist` folder after running `npm run build`.
- Build command: `npm run build`
- Publish directory: `dist`
- SPA redirects are handled in `netlify.toml`.
